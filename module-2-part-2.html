Declarative Components with JSX

With jQuery, you need to manually create DOM elements and manipulate the DOM. Rather than taking that imperative approach, React allows you to describe what your component is supposed to look like and the library takes care of rendering the component to the DOM.

Don't let the XML like syntax in your JavaScript frighten you. Essentially, JSX allows you to write HMTL in your JS code. This means we can write components and functions that treat HTML as a data type that gets returned and rendered to the DOM. Under the hood, Babel will transpile JSX into JavaScript code.
