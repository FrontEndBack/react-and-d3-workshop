Module 5: Final Project

The final project is the Alphabet example you saw earlier, but with a twist. We'll make it interactive. Animation happens as you type.

It looks simple, and the majority is simple. But there's a tricky final step that's more computer science than React or D3. I'll show you :)

We're going to build this example in 3 steps:

  1. Render the input box value as SVG text
  2. Add enter/exit/update animation
  3. Assign stable identifiers to letters
That last part is important, it lets React do enter/exit/update animations correctly. The meat of it also has the least to do with both React and D3.

Step 1: plain SVG text

Use this pen to build an App that takes some input and renders it as SVG text. I suggest building a Letter component for each letter. That will make the other steps easier.
