React and D3

Module 4: D3 + React Integration

Two Ways to Integrate

  • D3 does props
  • Blackbox components

D3 Does Props

  • React handles the DOM: in the render() method.
  • D3 functions calculate prop values: in updateD3() method.
  • Make sure D3 inner state stays in sync: call updateD3() when props change

See the Pen D3 Does Props by Freddy Rangel (@freddyrangel) on CodePen.

Blackbox Components

  • Wrap D3 code in React components: call render after React render.
  • Perfect for axes.
  • Quickly integrate D3 examples from the internet - everything in same function.
  • D3 handles DOM - d3.select React node.

See the Pen D3 Blackbox Components by Freddy Rangel (@freddyrangel) on CodePen.

Hybrid Approach

  • Define D3 objects in constructor.
  • Update them in updateD3.
  • Render with blackbox approach.

Simple animations

Change props through time -> animation. React is fast enough.

See the Pen Simple animation by Freddy Rangel (@freddyrangel) on CodePen.

Transition-based animation

Tiny bit trickier :)

  • Render from state, not props
  • Use d3.transition to change props directly
  • After transition, update React state

See the Pen Transition animation by Freddy Rangel (@freddyrangel) on CodePen.
