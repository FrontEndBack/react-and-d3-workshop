Integrating External APIs

In the previous section we learned about lifecycle methods. In this section we're going to use those lifecycle methods to fetch data from a REST API.

Since fetching data from a server is asynchronous, we don't know if and when we will receive that data. We probably want to display something to the user while we're waiting for that data. In that case, we want to use componentDidMount() to fetch the data only after our component has mounted to the DOM.
